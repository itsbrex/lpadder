# lpadder

![lpadder's banner](https://user-images.githubusercontent.com/59152884/177166304-5c79187c-4e43-4b37-8df3-9bd5d5130603.png)

lpadder, pronounced `el padder` or `launchpadder`, lets you play and edit covers that may require a [Launchpad](https://novationmusic.com/launchpad) or similar device.
